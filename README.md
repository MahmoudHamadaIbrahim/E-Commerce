# 🛒 FreshCart - Modern E-commerce Application

A high-performance, fully functional E-commerce web application built using **Angular** and **Tailwind CSS**. This project demonstrates advanced frontend concepts like state management with **Signals**, server-side rendering (**SSR**), and seamless API integration.

### 🚀 Key Features
* **User Authentication:** Secure Login/Register and token management.
* **Product Management:** Dynamic product listing with advanced filtering and pagination.
* **Shopping Cart & Wishlist:** Real-time updates using Angular Signals.
* **Optimized Performance:** Implemented SSR for better SEO and initial load speed.
* **Responsive Design:** Mobile-first approach using Tailwind CSS.
* **Interactive UI:** Smooth transitions and notifications using Ngx-Toastr and Swiper.js.

### 🛠️ Tech Stack
* **Framework:** Angular (v19+)..
* **Styling:** Tailwind CSS & FontAwesome.
* **State Management:** Angular Signals & Computed Properties.
* **Deployment:** Netlify (with SSR configuration).
* **Tools:** Ngx-Pagination, Ngx-Toastr, Swiper.
